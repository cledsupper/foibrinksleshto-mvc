Bem vindo ao FoiBrinks - Terminal de Atendimento do Vendedor

por Cledson Cavalcanti ("Leshto")


O FoiBrinks TAV é um projeto desenvolvido por mim no curso Bacharel em Sistemas de Informação da UFERSA - Campus de Angicos, sendo parte do processo avaliativo da componente curricular Programação Web.

O FoiBrinks atualmente se encontra na versão [DEVERIA APARECER O NÚMERO DA VERSÃO AQUI!!! OPORRA???]. É um projeto JavaEE que utiliza a biblioteca MariaDB para se comunicar com um banco de dados MariaDB/MySQL, o Bootstrap (versão:) 3 para o desenvolvimento das interfaces web, e o jQuery 3 para facilitar o uso do JavaScript (embora, na prática, eu tenha utilizado mais o JavaScript comum).
